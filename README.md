# 博客备份

1. 安装 hexo：
   1. sudo apt install nodejs 安装 Node.js
   2. sudo apt install npm 安装 npm
   3. npm install hexo-cli g 安装 hexo
2. 创建博文：
	hexo new post <title>
3. 命令：
   1. hexo -g 产生文件
   2. hexo -s 本机作为服务器，默认通过 http://localhost:4000/ 访问
   3. hexo -d 产生文件并部署
4. 配置文件
   1. _config.yml 里面配置了博客网站的一些内容，以及部署到哪里
   2. source/_posts 博客文章
